## Задание 1: Список всех Docker-контейнеров с датой
---
Написала простой скрипт, который:
1. Выводит все запущенные Docker-контейнеры
2. Добавляет к каждому строку текущую дату и время
3. Сохраняет результат в файл с названием containers-YYYY-MM-DD.txt
